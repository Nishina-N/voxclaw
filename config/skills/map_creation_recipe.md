# 地図生成レシピ：複数地点のアノテーション

## 概要
複数地点を一つの地図に収め、視認性を最適化するための手順書。

## 手順
1. **地点調査と座標取得**
   - 目的地のリストに対し、各地点の正確な緯度経度を検索。
   - `map_get_location` 等を活用し、確定させる。

2. **中心座標とズームレベルの算出**
   - 全地点を含むバウンディングボックス（最小/最大の緯度経度）を特定。
   - 中心点 = `(min_lat + max_lat) / 2, (min_lon + max_lon) / 2`
   - ズームレベルの調整: 
     - 全地点が適切に収まり、かつ詳細が確認できる距離感を見極める（例: ホテル5選なら 15.0〜15.5 程度）。

3. **Mapboxツール呼び出しの準備**
   - 運用ルールに従い、`map_get_mapbox_map` を**単一の呼び出し**で行う。
   - マーカーはカンマ区切りで指定: `pin-l-1+color(lon1,lat1),pin-l-2+color(lon2,lat2),...`
   - **注意点:** ズームレベルは全地点が見切れないよう、シミュレーションして設定すること。

4. **事後確認**
   - 生成された地図画像を確認し、端の地点が見切れていないかチェック。
   - 不合格の場合は中心座標とズームレベルを微調整し、再生成する。
